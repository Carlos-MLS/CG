A root é a pasta onde têm o build, engine, generator etc

1º COMPILAR (CMAKE): (na root)
  cd build
  cmake .. -B . && make
  ou usar a app CMake como nas aulas

2º GERAR PRIMITIVAS: (na build)
./generator plane 2 3 plane_2_3.3d
./generator box 2 3 box_2_3.3d
./generator sphere 1 10 10 sphere_1_10_10.3d
./generator cone 1 2 4 3 cone_1_2_4_3.3d

3º VER NO ENGINE (OpenGL): (na build)
   ./engine test_1_1.xml
   ./engine test_1_2.xml
   ./engine test_1_3.xml
   ./engine test_1_4.xml
   ./engine test_1_5.xml
   
   ou usar XML personalizado:
   ./engine ../config.xml



  O config.xml define a janela, camera e que modelos carregar.
  Controlos:
    WASD  - rodar camera
    Q/E   - zoom in/out
    C     - alternar camera fixa/orbital
    1     - modo solido
    2     - modo wireframe
    3     - modo pontos
    ESC   - sair

FORMATO .3D:
  Linha 1: número de vértices
  Restantes: x y z (um vértice por linha)

FORMATO XML (config):
  <world>
    <window width="512" height="512" />
    <camera>
      <position x="3" y="2" z="1" />
      <lookAt x="0" y="0" z="0" />
      <up x="0" y="1" z="0" />
      <projection fov="60" near="1" far="1000" />
    </camera>
    <group>
      <models>
        <model file="plane.3d" />
      </models>
    </group>
  </world>


ESTRUTURA:
  generator/
    main.cpp              - programa que gera os .3d
    primitives/           - plane, box, sphere, cone
  engine/
    main.cpp              - engine OpenGL/GLUT
    parser/xmlparser.h    - le o XML de config
    model/model.h         - carrega ficheiros .3d
  utils/
    point.h               - struct Point3D
    fileWriter.h          - escreve vertices para ficheiro


FICHEIROS DE TESTE (Fase 1):
  test_1_1.xml - cone com FOV=60
  test_1_2.xml - cone com FOV=20 (zoom)
  test_1_3.xml - esfera
  test_1_4.xml - box com clipping (far=3.5)
  test_1_5.xml - plane + esfera
  
Não esquecer:
Os XMLs de teste estão em test_files_phase_1/ e devem ser copiados
para a pasta build antes de executar o engine.
Para copiar: cp ../test_files_phase_1/*.xml .
